# Privacy-Focused Search Engines

Alternatives to Google that respect your privacy.

## Why Privacy Matters in Search

- Prevent search history tracking
- Avoid filter bubbles
- Reduce targeted advertising
- Protect personal information
- Maintain anonymity

## Top Privacy Search Engines

### DuckDuckGo
- No tracking or profiling
- Anonymous search
- Bang shortcuts (!g, !w)
- Clean interface
- Available on all platforms

### Startpage
- Google results without tracking
- Anonymous View feature
- EU-based (GDPR protected)
- No IP address logging
- Custom URL generator

### Brave Search
- Independent index
- No tracking
- Part of Brave browser
- Optional anonymous usage metrics
- Growing search results

### Qwant
- French privacy search
- EU-based servers
- No tracking
- News and social integration
- Music search feature

### Searx
- Meta-search engine
- Self-hostable
- Open source
- Highly customizable
- No tracking

## Features Comparison

| Engine | Tracking | Based | Results Source |
|--------|----------|-------|----------------|
| DuckDuckGo | None | USA | Bing + own |
| Startpage | None | Netherlands | Google |
| Brave | None | USA | Independent |
| Qwant | None | France | Own index |
| Searx | None | Varies | Multiple |

## Setting as Default

**Chrome/Edge:**
Settings → Search engine → Manage search engines → Add

**Firefox:**
Settings → Search → Default search engine

**Safari:**
Preferences → Search → Search engine dropdown

**Mobile:**
Similar steps in browser settings

## Search Tips

### DuckDuckGo Bangs
- `!g query` - Search Google
- `!w query` - Search Wikipedia
- `!yt query` - Search YouTube
- `!a query` - Search Amazon
- `!maps query` - Search Maps

### Advanced Search
Use quotation marks for exact phrases: `"cyber security tips"`

Filter by date, region, or safe search in settings

## Trade-offs

**Pros:**
✓ Privacy protection
✓ No filter bubble
✓ No ad targeting
✓ Fast and clean

**Cons:**
✗ Fewer personalized results
✗ Less location-specific
✗ Smaller index (some engines)
✗ Missing some Google features

## Browser Integration

**Brave Browser:**
- Built-in Brave Search
- Blocks trackers
- HTTPS upgrade
- Fingerprint protection

**Firefox:**
- DuckDuckGo optional
- Enhanced tracking protection
- Containers for isolation

## Mobile Apps

All major privacy engines have mobile apps:
- DuckDuckGo Privacy Browser
- Brave Browser
- Firefox Focus
- Qwant Mobile

## For Different Needs

**General Use:** DuckDuckGo
**Google Results:** Startpage
**Independent:** Brave Search
**European:** Qwant
**Tech-Savvy:** Searx (self-hosted)

Remember: No search engine is 100% private, but these are vastly better than mainstream options!
