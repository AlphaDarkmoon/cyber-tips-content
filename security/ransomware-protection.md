# Ransomware Protection

Protect your devices and data from ransomware attacks.

## What is Ransomware?

Ransomware is malicious software that encrypts your files and demands payment for their release.

## How Ransomware Spreads

- Phishing emails with attachments
- Malicious downloads
- Exploit kits on websites
- Remote Desktop Protocol attacks
- Infected USB drives

## Prevention Strategies

1. **Regular Backups** - Offline and cloud
2. **Keep Updated** - OS and software patches
3. **Antivirus** - Real-time protection
4. **Email Filtering** - Block suspicious attachments
5. **User Education** - Train on threats
6. **Network Segmentation** - Limit spread

## Warning Signs

- Files renamed with strange extensions
- Ransom notes appearing
- Unable to open files
- Suspicious processes running
- Unusual network activity

## If Infected

1. **Don't pay ransom** (no guarantee)
2. **Disconnect from network**
3. **Contact IT/security professionals**
4. **Report to authorities**
5. **Restore from backups**

## Best Practices

- Regular offline backups (3-2-1 rule)
- Disable macros in Office
- Use ad blockers
- Restrict user permissions
- Enable controlled folder access
