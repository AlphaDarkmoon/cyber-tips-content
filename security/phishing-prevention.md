# Phishing Attack Prevention

Learn to identify and protect yourself from phishing attacks.

## What is Phishing?

Phishing is a cyber attack that uses disguised emails, messages, or websites to steal sensitive information.

## Common Phishing Tactics

1. **Email Spoofing** - Fake sender addresses
2. **Urgent Messages** - Create panic to rush decisions
3. **Fake Links** - URLs that look legitimate
4. **Attachments** - Malicious files
5. **Impersonation** - Pretending to be someone you trust

## Warning Signs

- Spelling and grammar errors
- Generic greetings ("Dear Customer")
- Suspicious sender addresses
- Urgent or threatening language
- Requests for personal information
- Unexpected attachments
- Too-good-to-be-true offers

## How to Verify

1. Hover over links (don't click)
2. Check sender's email carefully
3. Contact company directly
4. Look for HTTPS and padlock icon
5. Verify requests through official channels

## Protection Steps

- Enable spam filters
- Use email authentication (SPF, DKIM)
- Install anti-phishing browser extensions
- Keep software updated
- Report phishing attempts
- Educate yourself regularly

## What to Do If Attacked

1. Don't click any links
2. Don't provide information
3. Report to IT/security team
4. Delete the message
5. Change passwords if compromised
6. Monitor accounts for suspicious activity
