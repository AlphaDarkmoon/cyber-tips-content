# USB Drive Security

Protect your data on portable storage devices.

## USB Security Risks

- Malware infection
- Data theft
- Lost/stolen drives
- Unauthorized access
- BadUSB attacks

## Encryption Options

### Windows: BitLocker To Go
- Built-in Windows feature
- AES-256 encryption
- Password or smart card
- Pro/Enterprise editions

### macOS: FileVault
- Encrypt entire drive
- Seamless integration
- Recovery key backup
- Fast encryption

### Cross-Platform: VeraCrypt
- Free and open source
- Multiple OS support
- Hidden volumes
- Plausible deniability

## Best Practices

1. **Always encrypt sensitive data**
2. **Use strong passwords**
3. **Scan for malware before use**
4. **Eject properly**
5. **Keep firmware updated**
6. **Label drives clearly**
7. **Regular backups**

## Physical Security

- Don't leave unattended
- Use keychain attachment
- Avoid public computers
- Destroy when disposing
- Use tamper-evident seals

## Safe USB Usage

**Before Plugging In:**
- Scan with antivirus
- Check for physical damage
- Verify source
- Use USB condom (data blocker)

**On Public Computers:**
- Avoid if possible
- Never save passwords
- Clear all data after
- Assume compromised

## Secure Deletion

When disposing:
1. Format multiple times
2. Use secure erase tools
3. Physical destruction (shredding)
4. Degaussing for high security

Remember: Treat USB drives like physical keys to your data!
