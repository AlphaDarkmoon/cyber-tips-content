# Cryptocurrency Security

Protect your digital assets and cryptocurrency investments.

## Wallet Types

### Hot Wallets (Online)
**Pros:** Convenient, easy to use
**Cons:** Vulnerable to hacks
**Examples:** Coinbase, Binance, MetaMask

### Cold Wallets (Offline)
**Pros:** Maximum security
**Cons:** Less convenient
**Examples:** Ledger, Trezor, paper wallets

### Best Practice
Keep majority in cold storage, small amounts in hot wallets for trading.

## Hardware Wallet Security

**Setup:**
1. Buy from official source only
2. Verify packaging is sealed
3. Generate seed phrase offline
4. Write seed phrase on paper
5. Never store digitally
6. Enable PIN protection

**Ledger/Trezor Tips:**
- Update firmware regularly
- Use passphrase (25th word)
- Test recovery before large amounts
- Store seed phrase in safe location
- Never share seed phrase

## Seed Phrase Protection

**Critical Rules:**
- NEVER share with anyone
- NEVER take photos
- NEVER store on computer/phone
- NEVER enter on websites
- Write on metal plate (fire-proof)
- Store in multiple secure locations

**If Compromised:**
Transfer funds immediately to new wallet

## Exchange Security

- Enable 2FA (authenticator app, not SMS)
- Use withdrawal whitelist
- Set up anti-phishing code
- Use strong unique password
- Withdraw to personal wallet
- Don't leave large amounts on exchange

## Transaction Safety

**Before Sending:**
- Double-check address
- Send test amount first
- Verify recipient
- Check network fees
- Confirm transaction details
- Save transaction ID

**Common Scams:**
- Address poisoning
- Fake apps/websites
- Phishing emails
- Ponzi schemes
- Fake giveaways
- Social engineering

## DeFi Security

**Smart Contract Risks:**
- Audit reports
- Team reputation
- Code verification
- Liquidity depth
- Community feedback

**Best Practices:**
- Start with small amounts
- Research thoroughly
- Use reputable platforms
- Understand risks
- Keep private keys safe

## NFT Security

- Verify contract address
- Use burner wallet for minting
- Be careful with wallet permissions
- Revoke unnecessary approvals
- Store valuable NFTs in hardware wallet

## Tax Considerations

- Keep detailed records
- Track all transactions
- Report accurately
- Use crypto tax software
- Understand local regulations

## Recovery Planning

**If Wallet Compromised:**
1. Stop using immediately
2. Transfer funds to new wallet
3. Review all transactions
4. Report to exchange/authorities
5. Create new secure wallet
6. Never reuse compromised wallet
