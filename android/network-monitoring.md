# Monitor Android Network Activity

Some apps may send data in the background without your knowledge.

## Safe Methods to Monitor Network Usage
- Use Android's built-in Data Usage settings
- Install NetGuard (no-root firewall)
- Review per-app network access

## What to Look For
- High background data usage
- Unknown apps connecting frequently

## Important Note
This guide focuses on **detection and awareness**, not interception.

## Recommended Apps
- NetGuard
- GlassWire (free tier)
