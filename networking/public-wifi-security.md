# Public Wi-Fi Security

Learn how to stay safe when using public wireless networks.

## Risks of Public Wi-Fi

- **Man-in-the-Middle Attacks** - Intercepted communications
- **Unencrypted Networks** - Data sent in plain text
- **Malware Distribution** - Infected networks
- **Snooping** - Others viewing your activity
- **Fake Hotspots** - Evil twin networks

## Before Connecting

### Verify Network Legitimacy
- Ask staff for official network name
- Avoid generic names like "Free WiFi"
- Check for password requirement
- Verify security settings
- Look for suspicious duplicate networks

### Device Preparation
- Update operating system
- Enable firewall
- Turn off file sharing
- Disable auto-connect
- Install antivirus software

## While Connected

### Safety Measures
1. **Use VPN** - Encrypt all traffic
2. **Access HTTPS sites only** - Look for padlock
3. **Avoid sensitive activities** - Banking, shopping
4. **Turn off sharing** - Network discovery off
5. **Forget network** - After use

### What to Avoid
- Online banking
- Shopping with credit cards
- Accessing work accounts
- Sending sensitive emails
- File sharing
- Auto-updates

## VPN Configuration

Most important step for public Wi-Fi:
1. Install trusted VPN app
2. Connect before accessing internet
3. Verify VPN is active (check IP)
4. Use kill switch feature
5. Choose nearby server

## Secure Alternatives

1. **Mobile Hotspot** - Use your phone's data
2. **Tethering** - Connect via USB/Bluetooth
3. **Cellular Data** - More secure than public Wi-Fi
4. **Portable Wi-Fi** - Personal hotspot device
5. **Wait** - Use trusted network later

## Hotel Wi-Fi Tips

- Ask for official network name
- Use separate login credentials
- Enable VPN immediately
- Use ethernet if available
- Disable auto-connect
- Forget network at checkout

## Coffee Shop Security

- Verify network with staff
- Use VPN
- Avoid financial transactions
- Keep sessions short
- Log out of all accounts
- Clear browser cache

## Airport Wi-Fi Caution

Airports are high-risk areas:
- Many fake hotspots
- Crowded networks
- International travelers targeted
- Use cellular data when possible
- Never access banking

## Business Travelers

- Use corporate VPN
- Follow company security policy
- Use dedicated work device
- Enable full-disk encryption
- Keep software updated
- Report security incidents

## Emergency Access

If you must use public Wi-Fi:
1. Enable VPN first
2. Access only HTTPS sites
3. Use two-factor authentication
4. Log out completely when done
5. Change passwords later
6. Monitor accounts for suspicious activity
