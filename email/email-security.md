# Email Security Best Practices

Protect your email account from threats and maintain privacy.

## Email Account Security

### Strong Authentication
- Unique, strong password (16+ characters)
- Enable two-factor authentication
- Use app-specific passwords
- Review account recovery options
- Security questions with unique answers

### Account Monitoring
- Review login activity regularly
- Check connected apps/devices
- Monitor for unusual emails sent
- Enable login alerts
- Check forwarding rules

## Recognizing Email Threats

### Phishing Red Flags
- Urgent or threatening language
- Unexpected attachments
- Requests for personal information
- Misspelled sender addresses
- Grammar and spelling errors
- Suspicious links

### Verification Steps
1. Hover over links (don't click)
2. Check sender's actual address
3. Contact company directly
4. Look for HTTPS in links
5. Verify through official channels

## Safe Email Practices

### Attachment Safety
- Scan with antivirus before opening
- Don't enable macros in documents
- Be cautious with ZIP files
- Verify sender before downloading
- Use online viewers when possible

### Link Safety
- Don't click suspicious links
- Type URLs manually instead
- Use link checkers (VirusTotal)
- Check destination before clicking
- Be wary of shortened URLs

## Email Encryption

### Options
**PGP/GPG** - End-to-end encryption
**S/MIME** - Certificate-based encryption
**ProtonMail** - Built-in encryption
**Tutanota** - Automatic encryption

### When to Encrypt
- Sensitive personal data
- Financial information
- Legal documents
- Medical records
- Confidential business

## Email Hygiene

### Regular Maintenance
- Unsubscribe from newsletters
- Delete old emails (retention policy)
- Archive important messages
- Empty trash/spam regularly
- Update email signature

### Organization
- Use folders/labels
- Set up filters automatically
- Star important emails
- Use search operators
- Keep inbox clean (inbox zero)

## Avoiding Spam

**Prevention:**
- Use email aliases for signups
- Don't post email publicly
- Use "+" trick (gmail+tag@gmail.com)
- Mark spam immediately
- Never respond to spam
- Use catch-all addresses carefully

**Spam Filters:**
- Enable aggressive filtering
- Create custom rules
- Whitelist important senders
- Report phishing attempts

## Mobile Email Security

**Settings to Enable:**
- Require passcode/biometric
- Auto-lock timeout
- Remote wipe capability
- Encrypted connections
- Disable auto-load images

**App Recommendations:**
- Native apps (updated regularly)
- Microsoft Outlook
- ProtonMail
- K-9 Mail (Android open source)

## Email Aliases

**Benefits:**
- Protect real address
- Track who shares/sells data
- Easy to abandon if spam
- Organize by purpose

**Services:**
- SimpleLogin
- AnonAddy
- Firefox Relay
- Apple Hide My Email
- DuckDuckGo Email Protection

## Business Email Security

### Additional Measures
- SPF records (verify senders)
- DKIM signatures (prevent spoofing)
- DMARC policies (reject unauthorized)
- Email gateway security
- DLP (Data Loss Prevention)
- Employee training

### Compliance
- GDPR (EU)
- HIPAA (Healthcare)
- SOX (Financial)
- Retention policies
- Encryption requirements

## Backup Important Emails

**Methods:**
- Export to PST/MBOX
- Use email backup software
- Forward to second account
- Save locally (encrypted)
- Cloud backup with encryption

**Frequency:**
- Critical emails: Immediately
- Important: Weekly
- General: Monthly

## Account Recovery

**Preparation:**
- Add recovery email
- Add phone number
- Print backup codes
- Document recovery process
- Keep info updated

**If Locked Out:**
1. Use account recovery
2. Contact support
3. Verify identity
4. Change password immediately
5. Review account activity
